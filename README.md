# python-job-scraper
Automated job scraping tool using Python (BeautifulSoup) with keyword filtering and CSV export.

This project scrapes job listings from a website using Python.

## Features
- Extracts job title, company, location, and apply link
- Filters jobs based on keyword (e.g., Python)
- Saves data to CSV file
- Adds timestamp for each job

## Technologies Used
- Python
- BeautifulSoup
- Requests

## Output
CSV file with structured job data
